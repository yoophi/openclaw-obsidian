## 클로드 코드 + 텔레그램 설치·활용 요약
*클로드 코드가 이제 텔레그램과 연동이 됩니다.* 집·사무실에 켜 둔 맥(예: 맥미니)에 클로드 코드를 설치해 두면 텔레그램(모바일·데스크톱)에서 원격으로 명령을 보내 맥의 컴퓨팅 자원(크롤링, 코드 작성, 파일 조작 등)과 연동된 에이전트를 24시간 활용할 수 있습니다. 설정 흐름은 깃허브에 올린 스킬(클로드 클로우)을 설치한 뒤 BotFather로 텔레그램 봇을 만들고 토큰을 등록·페어링한 다음, CLI에서 클로드 코드를 24시간 리스닝 모드로 실행해 채널 메시지를 받도록 하는 것입니다. 실제 활용 예로 구글 워크스페이스 연동으로 캘린더를 확인하고 메일 발송하거나, 파이썬 스크립트를 만들어 크론잡으로 정기 작업(예: 매일 주요 뉴스 10개 전송)을 예약하는 것을 시연합니다. *"24시간 입력을 대기하고 있습니다."*

현재 제한과 주의사항: 이 기능은 CLI(터미널)에서만 동작하며 데스크톱 앱·익스텐션 등 다른 클라이언트는 지원되지 않습니다(초기 버전 0.0.1). 클론잡·스케줄링으로 자동화는 가능하지만 세션 제한(예: 정기 리셋)과 권한 설정에 따라 원격에서 파일 삭제 등 위험한 명령도 실행될 수 있으므로 퍼미션을 신중히 설정해야 합니다; 특히 다른 메신저 연동(예: 카카오톡)은 법적·보안 리스크로 권장하지 않습니다. 초기 릴리스라 오류가 날 수 있고 향후 기능 확장(다른 플랫폼 지원 등)이 예상되니 CLI 사용에 익숙한 사용자에게 적합합니다.
