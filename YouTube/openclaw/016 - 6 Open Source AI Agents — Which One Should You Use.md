## Quick take
*There are now six serious open-source projects competing to be the brain of your personal AI assistant* — this video compares OpenClaw, NanoClaw, ZeroClaw, OpenFang, Hermes Agent, and Nanobot and tells you which fits different priorities (cost, security, open-model support, or fast setup). It’s a practical head-to-head that highlights architecture, sandboxing, model support, deployment footprints, and real-world tradeoffs so you can pick without deep extra research.

OpenClaw (TypeScript) is the original gateway-style platform with many channel plugins and a trusted single-operator model; NanoClaw (TypeScript) is minimalist and paranoid, running every agent session in ephemeral containers with a credential proxy but is tightly coupled to an Anthropic/Claude workflow. ZeroClaw (Rust) is the tiny, extremely low-footprint option (8.8 MB binary, runs on cheap Raspberry Pi) with strong encryption and kernel-level sandboxing; OpenFang (Rust) is the feature-heavy “agent OS” with prebuilt autonomous “hands,” the most extensive security stack (WASM sandboxing, taint tracking, Merkle audit trails) and fine-grained metering. Hermes Agent (Python) emphasizes a built-in learning loop and skills that improve over time, with an easy installer and multiple execution backends; Nanobot (Python) is the clean, research-friendly minimal agent that uses the lightLM abstraction for broad model support and the easiest setup.

Key recommendations mapped to common needs: if absolute cheapest and tiny hardware is your goal, choose ZeroClaw with a local Llama model. If you need the strongest, enterprise-grade defenses and prebuilt autonomous agents, pick OpenFang. If you want the fastest, lowest-friction install and readable code for modification or research, Nanobot is best. If you want a self-improving, skill-driven agent and an experience-focused workflow, Hermes Agent is unique. If you’re locked into Claude/Anthropic and want maximal isolation, NanoClaw’s container + credential-proxy model is the most paranoid. *If you want an AI that actually does things for you, not just answers questions* — this video shows which project will do that for your priority set.
