## Claude Code로 더 안전한 OpenClaw 대체 도구를 만든 사례

이 영상은 발표자가 Claude Code를 활용해 OpenClaw를 대체할 수 있는 더 안전한 도구를 만든 과정을 설명합니다. 출발점은 OpenClaw 같은 에이전트 시스템이 강력하지만, 권한 범위나 확장 구조 때문에 보안상 불안한 지점이 있을 수 있다는 문제의식입니다. 발표자는 이 점을 줄이기 위해, 더 통제 가능한 형태로 작업 흐름을 구성하고 위험 범위를 줄이는 방향의 대안을 실험합니다.

핵심 메시지는 단순히 OpenClaw를 비판하는 것이 아니라, **에이전트 시스템을 설계할 때 보안 경계와 권한 모델을 어떻게 다룰 것인가**에 있습니다. 발표자는 기능만 많은 시스템보다, 실제로 어떤 명령이 어디까지 실행되는지 더 명확하게 보이는 구조를 선호하며, Claude Code를 이용해 그런 형태를 구현하려고 합니다. 결과적으로 이 영상은 대체품 소개이면서도, 에이전트 설계에서 통제 가능성과 안전성이 얼마나 중요한지를 보여주는 사례입니다.
