## 오픈클로 3주 사용 요약

이 영상은 오픈클로(OpenClaw)를 2–3주 써 본 사용기로, 핵심 주장과 근거를 실사용 사례 위주로 보여 줍니다. 발표자는 오픈클로를 단순 채팅형 AI가 아니라 *“서버에서 24시간 일하는 AI 직원”*으로 규정하며 대화, 검색, 코딩, 파일관리, 브라우저 자동화 등 여러 도구를 하나로 통합해 텔레그램으로 지시만 하면 작업을 직접 실행한다고 설명합니다. 예시 증거로는 주가·암호화폐 백테스트(120일 이동평균 전술로 비트코인이 단순 보유 대비 6배 이상 수익), 자동화 스크립트가 실패했을 때 원격 PC를 경유해 자막을 추출한 사례, VNC로 유튜브 스튜디오에 접속해 채널 로고를 교체한 사례 등이 제시됩니다.

단점·주의점도 명확히 다룹니다: AI의 숫자 오류(할루시네이션)가 있어 금융 분석은 결과 검증 코드를 추가해 더블체크해야 하고, 복잡한 자동화는 호출량·비용이 커질 수 있습니다(예: 10분 스크립트로 API 호출 6,500회, 비용 약 4,500원; 복잡 분석 한 번에 만 원 안팎, 검증까지 하면 2만 원까지). 또한 여러 대의 컴퓨터를 연결해 쓸 때 보안 위험이 커지므로 비개발자는 주의하라고 권합니다. 실사용 팁으로는 토픽별 대화로 맥락 분리하기, 메모리 파일(memory.md)에 성향·규칙 저장하기, 작은 자동화부터 단계적으로 확장하기를 권장합니다. *“한 번 실패해도 포기하지 않고 다른 방법을 찾는다.”*
