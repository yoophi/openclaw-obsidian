## 자비스 프로토타입 오픈클로 2주 사용기 요약
영상은 맥 미니에 OpenClaw(오픈클로)를 설치해 '로컬 에이전트'로 24시간 돌려본 실사용기입니다. 제작자는 설치·채널 연동(텔레그램·왓츠앱·디스코드·iMessage 등)을 직접 보여주고, 오픈클로의 핵심은 미리 정의한 '스킬'과 여러 LLM을 중간 관리자처럼 조합해 작업을 지휘하는 구조라고 설명합니다. 그는 *"이 친구가 뭘 할 수 있는지"*를 테스트하면서 Gmail 정리, 캘린더 통합, 크론 자동화, 콘텐츠 기획 리포트 생성, 브랜드 링크 추출 같은 실제 워크플로우를 맡겼고 *"이게 에이전트다!"*라고 평가합니다.

주요 장단점과 실사용 데이터도 제시합니다. 모델 선택에 따른 차이를 상세히 설명하는데, 개발자 추천은 Claude Opus(4.6)이며 GPT 5.2는 안정적이지만 느리고, GPT 5.3 Codex는 기술적 응답이 많고 때때로 정밀도가 떨어진다고 합니다. API 비용 예시로는 Opus를 API로 돌릴 때 하루 약 $40(월 최대 $1,200) 수준이었고, 구독 연동(로컬 연동)은 비용·밴 위험의 트레이드오프가 존재합니다. 로컬 파일·서비스 제어와 메모리 접근으로 개인화는 강해지지만 권한을 많이 줄수록 보안·프롬프트 인젝션 위험도 커집니다. 하드웨어 측면에선 CPU·메모리 병목이 문제였고(병렬 작업 중 불안정), 맥 미니 대신 더 강력한 기기(Mac Studio 등)를 고려할 만하다고 결론짓습니다. 최종 평가는 "생각보다 쓸 만하다, 단 비용·보안·리소스 관리는 필수"입니다.
