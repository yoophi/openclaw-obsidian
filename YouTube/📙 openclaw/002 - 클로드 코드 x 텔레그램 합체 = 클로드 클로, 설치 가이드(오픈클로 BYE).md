## 클로드 코드 × 텔레그램 설치 가이드 요약

핵심: 클로드 코드(Claude Code)를 텔레그램과 연동하면 집·사무실에 켜 둔 PC(발표자는 맥미니)에 상주하는 에이전트를 폰·PC 어디서든 텔레그램으로 제어할 수 있습니다. 연동된 에이전트는 *클로드 코드는 24시간 입력을 대기하고 있습니다.* 상태로 메시지를 받자마자 작업을 수행하고, *맥미니의 자원을 사용할 수 있다*—코드 실행, 크롤링, 구글 워크스페이스 조작, 메일 발송, 파일·바탕화면 조작 등 컴퓨팅 자원과 권한이 허용되는 범위에서 직접 작업합니다. 이 기능은 CLI(터미널) 환경의 클로드 코드에서만 동작하며 데스크톱 앱·익스텐션·코워크 등에서는 아직 지원되지 않습니다.

설정 흐름(요약된 단계): 1) 클로드 코드가 설치된 상주시킬 PC에서 저장소(작성자는 "클로드 클로우 셋업" 스킬)를 가져와 설치·스킬 활성화. 2) 텔레그램에서 BotFather로 새 봇을 만들고 봇 토큰을 발급받아 클로드 코드에 등록(허용 사용자 설정 포함). 3) 발급된 채널/페어링 코드를 클로드 코드에 입력해 봇과 페어링. 4) 터미널에서 클로드 코드를 24시간 대기(listening) 모드로 실행할 때 필요한 플래그(예: permissions 등)를 추가해 항상 채널 메시지를 수신하도록 설정. 설치 과정 중 마켓플레이스 플러그인(텔레그램 플러그인 오피셜 v0.0.1) 설치, 토큰 저장, 권한·허용 사용자 구성 등 클로드 코드가 안내해 주는 대로 진행하면 됩니다. 영상은 실시간 데모(메시지 주고받기, 구글 캘린더 조회·메일 발송 예시, 크론잡으로 자동화 실행 등)를 보여 줍니다.

주요 기능·활용 예시: 텔레그램에서 명령 보내면 PC의 에이전트가 CLI로 코드 작성·실행, 웹서치·크롤링, GWS(구글 업무) 액세스, 메일 전송 등을 처리합니다. 크론잡을 걸어 정기 작업(예: 매일 아침 뉴스 10개 요약 전송, 예약 메일 발송, 크론잡 리스트 리포트)을 자동화할 수 있으며, 세션·요금제 한도(예: 세션 리셋 주기)가 있으므로 자동화 스케줄링으로 사용량을 관리하는 꿀팁도 제시합니다. 발표자는 카카오톡 연동 같은 비공식적·잠재적 불법성·보안 위험이 있는 사용은 추천하지 않으며, 특히 24시간 상주시킨 에이전트에게 광범위한 권한을 부여하면 파일 삭제 등 위험 행동을 할 수 있으므로 권한 관리를 권합니다.

제약·주의사항: 현재 버전(초기 0.0.1)은 CLI 전용이며 오류가 발생할 수 있고(초기 설치 시 몇몇 에러), PC가 꺼지면 동작하지 않습니다. 권한(permission) 옵션을 통해 기능 범위를 제한해야 하며, 허용 사용자 설정으로 접근을 제어해야 합니다. 클로드 코드의 세션 제한(주기적 리셋)과 가격제(플랜별 사용 한도)를 고려해 사용 패턴을 설계해야 합니다. 영상은 초심자 대상 설치·페어링·데모·크론잡 예제까지 차근차근 보여 주므로, CLI 기반 자동화와 원격 에이전트 운영 방법을 직접 따라 해 보고 싶은 사람에게 실용적입니다.
