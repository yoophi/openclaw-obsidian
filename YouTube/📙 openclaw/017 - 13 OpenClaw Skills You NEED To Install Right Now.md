## 13 OpenClaw Skills You NEED To Install Right Now

This video tests hundreds of OpenClaw skills and highlights the 13 that materially changed the creator’s workflow: they cut monthly API costs from roughly $1,200 to about $40, automated large parts of content and product funnels, and set up agents that learn over time. The presenter warns the ecosystem is noisy and has had malicious uploads, so run basic due diligence when installing community skills. He groups the picks by impact: content automation and growth, agent-level tooling (self-upgrading, memory, token efficiency), integrations (email, calendar, chat, browser), and research/analytics helpers.

Top picks and why they matter: Larry — an agent-built Tik Tok slideshow engine that auto-generates six portrait images with locked scene architecture, precise text sizing/placement rules, and posts drafts so one human step (add trending audio) publishes; it also tracks funnel metrics across views, downloads, trials, and subscriptions to tell you what to fix. *Larry generated over 8 million views on Tik Tok.* Capability Evolver — a meta-skill that reviews failures and rewrites the agent’s code so *your AI agent debugs itself and upgrades itself automatically.* QMD (token killer) indexes local docs with BM25 + vector search so the agent fetches only the paragraphs it needs, cutting token use up to ~70%. Anti-AI-Slop Humanizer removes common AI-writing “tells” and a code-specific variant strips overengineered patterns from generated code. Exa AI Web Search restores web access for agents without extra API cost. GOG unifies email, calendar, drive, docs and sheets under a single CLI so the agent can summarize, draft replies, block time, and update spreadsheets. The X (Twitter) Research skill offers filtered signal extraction, cost transparency per query, and watchlists. Bite Robot provides persistent, queryable project memory that survives sessions. WhatsApp CLI, Playwright browser automation, Mission Control (daily dashboard/briefing), and an X Impact Checker (pre-post tweet scoring against the known recommendation signals) round out the list; install counts and usage notes are given for many.

Practical install and safety notes: some skills need API keys (image generation, web search alternatives) or “posters” and analytics for automated publishing; QMD and Bite Robot dramatically reduce repeated reading of large documents; Capability Evolver has a review mode and a continuous “mad dog” mode but requires backup/versioning to avoid risky recursion. The presenter shares specific micro-rules learned from iteration (e.g., font should be ~6.5% of image height for phone readability; place text ~30% from the top to avoid overlays) and recommends combining complementary skills (for example, GOG + X Research + Mission Control to automate morning briefings). Overall takeaway: many high-impact automations are already available as community skills, but treat them like code—inspect, sandbox, and back up before giving broad permissions.
