## 한글 요약
Pi coding agent — a minimal, highly extensible coding assistant

The video’s main claim is that the Pi coding agent is intentionally minimal yet powerful: it ships with a very small system prompt and just a few built-in tools — *“it only comes with read, write, edit, and a bash tool.”* That bash tool is the workhorse because a shell lets the agent control the host machine, call CLIs, fetch web data, run scripts and parse JSON, giving it broad capabilities while keeping the agent’s context uncluttered; as the presenter puts it, *“you get an agent that has a context window that's not cluttered up.”*

Support for extensibility is the other headline: Pi uses lazily loaded Agent Skills (skill directories and skill MD files) and a first‑party Extensions system so you can add features (plan mode, extra slash commands, UI updates), share or install packages from an extensions marketplace, or add web access and sub‑agents. The presenter demos noncoding workflows — e.g., a seven‑day Apple/NVIDIA stock analysis where the agent performed web research, wrote and executed temporary scripts, fetched prices and computed performance — and notes Pi can be configured per project or globally and paired with other agents like Codex or Claude for different tradeoffs between out‑of‑the‑box tools and customizability.
