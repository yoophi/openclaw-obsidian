## Using Claude Code remote control

This clip shows how to pick up a Claude Code session from another device: from your project directory run `claude remote control` to start a session and get a URL (press the space bar to show a QR code for phones). If you’re already in a session, typing `/remote control` shows the link; because sessions are tied to your account you can also open the session from the mobile app or web without clicking the provided URL.

The video stresses that the work stays local while you use remote devices: *"Everything is still running on your machine. Nothing is moved to the cloud."* You get the same tools, servers, configurations, and file access, and *"The conversation then stays in sync across devices."* You can enable remote control by default in `/config`. A spawn mode lets Mac, team, and enterprise users create new remote sessions and spawn multiple instances; the feature is rolling out to pro and max plan users.
