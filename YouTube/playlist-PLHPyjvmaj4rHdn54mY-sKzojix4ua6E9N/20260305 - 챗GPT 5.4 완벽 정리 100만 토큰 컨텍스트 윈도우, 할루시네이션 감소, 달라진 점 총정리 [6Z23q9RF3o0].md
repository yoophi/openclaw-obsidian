## 한글 요약
GPT 5.4 한눈 정리

GPT 5.4는 기존 5.3(코딩 중심)과 5.2(사고 중심)를 통합해 코딩 능력과 추론 능력을 모두 끌어올린 모델입니다. 핵심 변화는 더 빠르고 정확해진 응답, 할루시네이션 감소, 자동 라우팅(간단한 응답은 인스턴트, 복잡한 사고는 씽킹으로 자동 분배)이며, 씽킹에는 라이트·스탠더드·익스텐디드·헤비(프로 전용) 등 난이도별 모드가 있습니다. 실사용 데모에서 PRD 작성→코드 생성→MD 파일 실행까지 자동 스캐폴딩이 가능했고, 작성된 앱(예: AI 식단 큐레이션)과 PRD 결과물이 이전 버전보다 더 상세하고 실무적이라 평가했습니다. *GPT 5.4가 드디어 출시가 되었습니다.*

가장 주목할 점은 컨텍스트 윈도우 확장과 속도 옵션입니다. UI에서는 5.4 기준으로 기본 256k 토큰 지원(플러스/비즈 등등 계층별로 차등 제공)이며, 코덱스와 API에서는 실험적으로 100만 토큰까지 설정할 수 있다고 발표했습니다. 또한 새로 도입된 페스트 모드는 응답 속도를 약 1.5배로 올리지만 토큰 소모가 빨라지고(특히 API 사용 시 과금 영향) 큰 문맥을 길게 쓰면 소모가 더 커집니다. 스프레드시트·슬라이드 제어용 스킬도 제공되어 문서·프레젠테이션 생성 워크플로우가 강화되었고, 저자는 클로드 코드와 비교해 기능 차이는 거의 없지만 출력 스타일·디자인 취향에서 차이를 느꼈다고 합니다. *컨텍스트 윈도우를 100만 토큰을 주어진다고 공식 발표를 했습니다.*
