## Summary

This clip demonstrates a multi-agent "CEO + board" system that elevates agents from low-level workers to strategic decision-makers by combining three technical levers: large 1,000,000‑token context windows with steady pricing, a customizable agent harness (extension) that orchestrates parallel agents and enforces constraints, and persistent agent expertise files (long scratchpads) that accumulate domain memory. The harness takes a templated brief (stakes, constraints, key questions) as input, spins up role‑based agents (revenue, compounder, product strategist, contrarian, moonshot, etc.) that respond in parallel under a single CEO controller, and produces a final memo with a decision map, stances, tensions, next actions, SVG visuals and a natural‑language summary. Runs shown: a marketing/platform brief that surfaced a retention problem before recommending a shorts channel, and an acquisition brief that produced a 5–1 board recommendation to accept a $12M offer at 11× ARR with specific conditions; the system stops when time/budget constraints are hit and asks each agent for a final position.

The video emphasizes concrete engineering patterns: validate and template briefs so agents get structured inputs; override and specialize each agent’s system prompt (temperament, heuristics, skills); record full conversations for observability; and persist expertise files so agents accumulate long-term, domain-specific knowledge (examples: multi‑thousand token expertise logs). Parallel, opinionated agents create adversarial debate that surfaces trade‑offs and blind spots, increasing the memo’s signal for high‑leverage decisions. *uncertainty in and we have decisions out.* *How much is a high leverage decision really worth to you?*
