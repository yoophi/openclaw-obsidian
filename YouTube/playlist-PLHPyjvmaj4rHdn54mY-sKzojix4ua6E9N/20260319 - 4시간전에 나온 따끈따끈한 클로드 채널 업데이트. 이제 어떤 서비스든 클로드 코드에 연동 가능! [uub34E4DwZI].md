## 클로드 코드 채널(Claude Code Channels) 업데이트 요약

이번 릴리스는 MCP를 통해 외부 서비스가 '실행 중인' 클로드 세션으로 직접 프롬프트나 이벤트를 푸시(push)할 수 있게 하는 기능을 공개했다는 핵심 변화입니다. 데모는 텔레그램 플러그인으로 진행되며, 흐름은 플러그인 설치 → 채널 활성화 → BotFather로 봇 생성 및 HTTP API 토큰 복사 → 클로드에 토큰 등록(telegram:config) → 페어링/권한(allowlist 등) 설정 순입니다. 이로써 기존처럼 폴링으로 데이터를 끌어오는 방식이 아니라 MCP에서 세션으로 바로 인젝트할 수 있고, 제시된 문서에도 *이미 실행 중인 세션에 채널로 이벤트를 푸시할 수 있다.*고 명시되어 있습니다.

실제로 클로드는 텔레그램에서 보낸 메시지를 받아 응답하고, 예시로 "HTML/CSS/JS로 클로드 코드 사용 가이드 사이트를 만들어 달라"는 요청을 처리해 페이지 코드를 생성하는 작업을 시작했습니다. 엔트루픽 쪽은 텔레그램·디스코드 외에 인터페이스를 개방해 원하는 서비스로 확장 가능한 'build your own channel'을 제공하고 있어, 서버에서 직접 특정 사용자의 세션에 프롬프트를 쏘는 맞춤형 통합이 가능해졌습니다. 현재는 프리뷰·실험 단계라 변화가 있을 수 있다는 점을 유의해야 합니다. *프롬프트를 쏠 수 있는 상황이 됐다.*
