## Claude Code: what the new Skill Creator actually does

The video’s core claim is that Claude Code’s new Skill Creator turns skills from black-box prompts into testable, improvable components: it writes evals, runs benchmarks (AB tests, multi-agent parallel runs), tracks regressions and measures trigger reliability so you know whether a skill truly improves outputs. The presenter demonstrates concrete outcomes (token counts, pass rates, latency) and shows the tool catching cases where a skill either helps or is rendered unnecessary by model upgrades. *“skills are just a text prompt that tells Claude code how to do a certain thing in a certain way.”*

It also explains two evaluation paths tied to skill type: capability uplift skills (improve model performance on a task) are benchmarked with skill-vs-no-skill AB tests to detect regressions as models change; encoded-preference skills (workflows and ordering) are evaluated for fidelity to the intended steps and outputs. The Skill Creator tunes skill descriptions to improve triggering, can run many tests at once, and requires no code: install the official plugin, restart Claude Code, then invoke the skill creator to generate, modify, and eval skills. *“is my skill actually doing what it's supposed to do”*
