## 한글 요약
요약
이 영상은 클로드의 새 기능 디스패치(Dispatch)를 소개합니다. 핵심은 *폰에서 컴퓨터의 클로드한테 채팅으로 일을 시키는 기능입니다.* 즉 모바일에서 메시지로 지시를 보내면, 집이나 사무실에 켜진 클로드(코워크)가 컴퓨터의 파일과 앱을 직접 열어 수정·비교·일괄 처리 같은 작업을 실행하고 결과를 폰에서 확인하게 해 줍니다. 동작 흐름은 폰에서 지시 → 원격 컴퓨터의 클로드가 실행 → 폰에서 결과 확인이며, 대화 기록을 이어 받아 이어서 작업할 수 있습니다.

설정은 데스크톱과 아이폰 앱을 최신 버전으로 업데이트한 뒤 코워크 패널의 디스패치 QR코드를 폰으로 스캔하면 끝입니다(*QR코드 한 번이면 설정 끝입니다.*). 제약은 프로 또는 맥스 요금제가 필요하고 컴퓨터가 켜져 있고 절전 모드가 아니어야 하며(노트북 덮으면 멈춤), 완료 알림이 따로 없고 클로드가 먼저 연락하지 않는다는 점입니다. 요약하면, 폰으로는 못 하는 ‘컴퓨터 파일 직접 조작’을 원격으로 처리해 주는 기능이지만 항상 켜진 컴퓨터와 수동 확인이 전제입니다.
