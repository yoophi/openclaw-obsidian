## 소놀봇 2세대(클로드 코드 + 코덱스) 요약

텔레그램에서 클로드 코드와 코덱스 양쪽을 동시에 쓸 수 있게 만든 소놀봇 2세대를 시연하는 영상입니다. 여러 개의 봇(시연에선 총 7개)을 병렬로 돌려 “에이전트”처럼 각각 독립 작업을 맡겨 한 번에 다수 작업을 처리했고, 슈퍼마리오 월드 1-1 리서치와 이미지 분석·저장, 게임 소스 FTP 배포(플레이 가능한 URL 제공), 랜딩페이지·3D 패럴랙스 사이트 제작, 자동 이미지 생성과 음향 생성까지 실무 흐름을 보여줍니다. 제작 속도와 응답성이 크게 개선되어 *“15분 만에 리서치가 완료되었고”* 실제 사용자가 외부에서 원격으로 지시해도 *“한 번에 굉장히 많은 작업을 진행할 수 있어요.”* 라는 결과를 확인할 수 있었습니다.

설치와 운영 흐름도 상세히 안내합니다. 제공된 폴더의 셋업 파일을 관리자 권한으로 실행해 컨트롤 패널을 만든 뒤, 텔레그램 BotFather로 봇을 생성해 토큰을 등록하고 전역 허용자(ID)를 설정한 다음 데몬을 시작하면 됩니다. 코덱스는 WSL·파이썬·노드·코덱스 CLI 등 사전 구성 요소가 필요하고 클로드 코드는 WSL 불필요하지만 사전 로그인 상태여야 합니다. 각 봇은 메인 모델(예: GPT‑5.3 코덱스)과 대화용 리라이터 모델을 분리해 설정할 수 있고, 작업 단위마다 고유한 에이전트 MD 지침 파일을 주입해 깨끗한 컨텍스트로 진행합니다. 스킬 검색 도구(SP search)는 20만개 이상 후보를 자동 탐색하지만 보안 정책으로 위험한 스킬 설치는 보류 처리해 안전성을 확보했습니다.
