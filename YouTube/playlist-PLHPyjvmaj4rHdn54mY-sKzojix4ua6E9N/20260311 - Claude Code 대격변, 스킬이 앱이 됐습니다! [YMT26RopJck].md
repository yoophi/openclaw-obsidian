## 한글 요약
Claude Code 스킬 2.0 요약
클로드 코드의 스킬 2.0은 기존의 '참고 문서형 스킬'을 격리 실행 가능한 소프트웨어 컴포넌트로 진화시킨 업데이트로 핵심은 컨텍스트 효율화와 자동화된 품질관리입니다. YML 프론트메타로 스킬별 훅·분류(classification)·라이프사이클을 선언해 컨텍스트 포크로 격리 실행하고, 스킬 크리에이터는 스킬 생성·리로드·테스트(에발스)를 자동화해 세션을 다시 시작하지 않고 즉시 변경을 반영합니다. 이로써 토큰·컨텍스트 사용량을 줄이고 개발→검증→배포 사이클을 단축해 *스킬이 소프트웨어가 됐다.* 

주요 구성 요소 다섯 가지는 (1) 격리된 컨텍스트 실행(컨텍스트 포크), (2) 스킬별 훅과 라이프사이클 제어, (3) AI에 의한 자동 품질검증(에발스), (4) 스킬 자동 생성 도구(스킬 크리에이터), (5) 분류 기반 수명·워크플로 관리입니다. 실사용 효과로 발표자들은 스킬 제작 시간이 기존 30분–2시간에서 10–30분으로 단축됐고, 수정 시 세션 재시작 불필요·자동 테스트·정량적 리포트 제공 등으로 비개발자도 안전하게 스킬을 만들고 운영할 수 있다고 보고합니다. 이러한 변화는 모델 성능 자체보다 ‘컨텍스트 엔지니어링·워크플로 설계’가 AI 네이티브 경쟁력이라는 주장과 맞닿아 있습니다.
