## 한글 요약
n8n 2026 마스터클래스 — 핵심 요약
이 8시간짜리 강의는 n8n의 기초부터 실전 배포까지 단계별로 안내한다. 인터페이스(워크플로우·크레덴셜·익스큐션), 시작점(수동·스케줄·웹훅·폼·메시지), 데이터 타입·JSON 처리, API(HTTP 요청)와 인증·헤더·바디 구성 같은 핵심 개념을 차근히 설명한 뒤 *이미지 생성 API·웹 크롤러·회의 전사 도구·LLM*을 연결해 실제 파이프라인을 만든다. 강의는 예시 중심인데, 대표적으로는 (1) 유튜브 채널 모니터 → 대본 추출 → AI 요약 → 구글 문서·이메일 리포트 자동화, (2) 텍스트→이미지/이미지→이미지 변환 후 구글 드라이브 업로드, (3) 회의 녹취 자동 전사 → 요약 → 아카이브·초안 생성 같은 실전 프로젝트를 따라 만든다. 강사는 *“자동으로 내가 뭔가 개입을 안 하더라도 유연하게 흐를 수 있는 시스템을 구축 가능”*이라고 반복해 강조한다.

핵심 실무 팁: API 호출은 HTTP 노드로 세부 엔드포인트·메서드·헤더·바디를 정확히 세팅하고, 대용량·바이너리 파일은 다운로드→클라우드 스토리지 업로드로 처리하라. 로그·에러(재시도·continue 옵션·에러 트리거) 설계와 테스트/버전 관리, 그리고 스키마(테이블) 설계가 자동화 성공률을 좌우한다. LLM 응용은 요약·프롬프트 설계·출력 포맷(구조화된 JSON)으로 품질을 끌어올리고, RAG(문서 청크→임베딩→벡터 DB→검색→LM 결합) 개념을 도입하면 최신·정확한 답변을 만들 수 있다. 끝으로 강사는 *“복습과 반복만이 출력을 향상시킬 수 있다”*며 연습과 작고 반복적인 자동화부터 확장하길 권한다.
