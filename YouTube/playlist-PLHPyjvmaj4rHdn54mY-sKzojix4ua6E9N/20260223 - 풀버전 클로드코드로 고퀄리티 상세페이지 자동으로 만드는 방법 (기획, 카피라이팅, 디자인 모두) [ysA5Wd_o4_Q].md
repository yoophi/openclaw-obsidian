## 요약

영상은 클로드 코드(Claude Code)를 에이전트 오케스트레이션으로 활용해 상세페이지(랜딩페이지)를 100% 자동 생성하는 방법을 단계별로 시연한다. 핵심 흐름은 정보수집 → 리서치 → 카피라이팅 → 디자인 → 프롬프팅(이미지 생성)으로 구성된 에이전트 팀을 만들고, 클로드 코드의 공식 서브에이전트·스킬 구조(MD 파일·스킬 폴더 등)를 준수하는 것이다. 최종 산출물은 제미나이 3 프로 API로 생성한 약 13개의 이미지를 파이썬 스크립트로 병합해 PNG 또는 PDF 형태의 긴 상세페이지로 내보내는 방식이며, 시스템 프롬프트 설계와 .env(및 API 키) 세팅이 필수라고 설명한다.

데모에서는 먼저 미완성(만화풍·비정형 레이아웃) 결과를 받고, 가로세로 1200 고정·모델 비중 증가·제품 디테일 강화 등 반복 지시로 품질을 개선한 뒤 최종 샘플(화장품, 유모차)을 생성해 실제로 '초안으로 바로 쓸 만한 수준'까지 끌어올리는 과정을 보여준다. 실무 팁으로는 클로드 코드 가이드라인에 맞춘 에이전트·스킬 구성, 13개 섹션 프롬프트 적용, 제미나이 API 키를 .env에 정확히 넣는 세 가지를 강조하며, 자동 출력은 초안으로 활용하되 최종 라이브 전에는 사람의 검수·최적화가 필요하다고 권장한다.
