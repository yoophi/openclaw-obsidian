영상은 스킬의 두 가지 주요 유형(모델 능력을 보완하는 '능력 향상' 스킬 vs. 워크플로우·선호도를 고정하는 '선호도 인코딩' 스킬)과, 기본 모델이 향상되면 일부 스킬은 은퇴할 수 있다는 실무적 위험을 설명합니다. 데모 벤치마크 결과로는 스킬 활성화 시 성공률(영상 내 수치 예: 약 13.5% 증가), 작업 완료 평균 시간 22% 단축, 토큰 사용량 약간 증가 등이 보고되며, 평가 리포트에서 각 출력의 점수·타이밍을 확인하고 피드백 파일을 내려받아 개선안을 적용하는 흐름까지 시범 보였습니다. 전반적으로 자주 쓰는 복잡한 스킬에 적용하면 신뢰성과 성능을 객관적으로 검증·향상시킬 수 있다는 결론입니다.
