# coding agent 표준 보고 프롬프트

생성일: 2026-04-09

## 목적
coding agent를 작업자이면서 보고자 역할로 함께 쓸 때, 텔레그램으로 일관된 형식의 진행 보고를 보내기 위한 표준 프롬프트 모음.

## 1) 완료 보고용
```text
텔레그램으로 아래 형식으로 작업 완료 보고 보내:

[작업 완료]
- 작업명:
- 결과:
- 변경 사항:
- 검증:
- 관련 링크:
- 다음 작업:

가능하면 5~10줄 안으로 간결하게 써.
```

## 2) 진행 중 보고용
```text
텔레그램으로 아래 형식으로 진행상황 보고 보내:

[진행 상황]
- 현재 작업:
- 지금까지 한 일:
- 현재 상태:
- 남은 작업:
- 예상 다음 단계:

짧고 명확하게, 불필요한 배경설명 없이 써.
```

## 3) blocker 보고용
```text
텔레그램으로 아래 형식으로 blocker 보고 보내:

[이슈 / blocker]
- 막힌 작업:
- 원인:
- 영향 범위:
- 시도한 것:
- 필요한 결정 또는 도움:
- 임시 우회 가능 여부:

핵심만 빠르게 판단할 수 있게 써.
```

## 메모
- 이후 필요하면 SpecRail 전용 버전으로 더 구체화할 수 있음
- 한 줄 요약 템플릿, 상태판 형식도 이 문서를 바탕으로 파생 가능
